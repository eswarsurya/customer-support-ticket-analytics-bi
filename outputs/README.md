# Public Output Evidence

This folder contains public-safe output evidence for the Customer Support Ticket Analytics BI project.

## Files

- `dataset_profile.csv` summarises the dataset and confirms the raw dataset is not uploaded.
- `status_distribution.csv` summarises ticket status.
- `priority_distribution.csv` summarises ticket priority.
- `source_distribution.csv` summarises support contact channels.
- `first_response_sla_distribution.csv` summarises first-response SLA performance.
- `resolution_sla_distribution.csv` summarises resolution SLA performance.
- `support_level_distribution.csv` summarises Tier 1 / Tier 2 support routing.
- `top_topics.csv` lists the most common support topics.

## Privacy Note

Direct ticket IDs, agent names, exact timestamps, and location coordinates are excluded from the public sample.
