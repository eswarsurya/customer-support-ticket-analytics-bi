# Data Notes

The original project dataset is an Excel workbook of customer support tickets. The raw workbook is not uploaded publicly because it includes ticket IDs, agent names, exact timestamps, and coordinate fields.

## Public Sample

`public_sample_support_tickets.csv` is a small public-safe sample for repository review. It removes direct ticket and agent identifiers, removes coordinates, and converts exact timestamps into month-level values.

The sample is intended to show the structure behind the dashboard, not to replace the original analysis dataset.
