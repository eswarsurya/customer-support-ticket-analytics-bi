# Dashboard Evidence

The original project includes Power BI and Tableau dashboard files prepared locally:

- `ServiceDeskDashboard.pbix`
- `ServiceDeskDashboard.twbx`

These binary dashboard files are not uploaded through this text-based GitHub connector. The public repository still includes the dashboard logic through safe data samples, output summaries, report notes, and dashboard output documentation.

## Dashboard Focus

The dashboard is designed to review:

- Ticket status and backlog movement
- Priority mix
- Support source channels such as Email, Chat, and Phone
- First-response SLA performance
- Resolution SLA performance
- Support tier distribution
- Most common customer support topics

## Public Sharing Note

Before uploading dashboard binaries publicly, the workbook should be checked for embedded raw data, agent names, ticket IDs, and other fields that should not be public.
